==Description==

Make your text widget accept shortcodes just by installing this plugin. You don't have to do anyting, just install it and you're good to go.

==Installation==

1. Upload Atomic.Penguins.Shortcode.on.Text.Widget.zip to wp/content/plugins directory

2. Activate the plugins through the 'Plugins' menu in Wordpress Admin

3. You're done.
